Flutter GPS Tracker

This repository contains a cross-platform starter app for Android and iOS GPS tracking built with Flutter.

Implemented Features

  • Live location tracking with start/stop controls
  • Distance and elapsed time counters
  • Polyline route rendering on OpenStreetMap
  • Android and iOS location permission setup

Run Locally

From flutter_gps_tracker/:

flutter pub get
flutter run

Source

See app code in flutter_gps_tracker/lib/main.dart.
